# Aegis Notes

## Work Record Usage

- `docs/aegis/work/*` is temporary working memory, not a permanent process log.
- During active work, checkpoint files may exist while they help coordination.
- At closeout, clean the work directory yourself.
- Replace stale process files with one short `SUMMARY.md`.
- Keep durable future-facing decisions or task scopes under `docs/aegis/specs/`.
- Do not preserve long completed checklists, raw evidence dumps, or expired
  scratch notes after the task has moved on.
